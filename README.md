# Salary Management Platform

Salary management software for an HR manager handling 10,000 employees across multiple countries.

## Stack

- Backend: Node.js, Express, TypeScript, Prisma, PostgreSQL
- Frontend: React, Vite, TypeScript
- Validation: Zod
- Tests: Vitest, Supertest

## Monorepo

- `apps/api`
- `apps/web`
- `docs`
